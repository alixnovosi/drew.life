Twitter Bots - #NaBoMaMo 2016

  1. dirtyunix_bot

    A bot combining command-line utils in ways that sound as dirty as possible. Language: Python 3

  2. isthisska_bot

    A bot pulling random album art from MusicBrainz (API: https://musicbrainz.org/doc/Development/XML_Web_Service/Version_2) and asking if it's ska. Language: Python 3

  3. randweather_bot

    A bot describing the weather in a random city. Pulls from https://openweathermap.org/ and a manually-created zip code list. Language: Python 3

  4. weatherlies_bot

    A bot lying about the weather in a random city. Pulls from same sources as randweather_bot. Language: Python 3

  5. fantasymetro_bot

    A bot generating plausible but imaginary metro systems for cities. Needs a lot of manual work to generate for a given city, unfortunately. Language: Python 3

  6. not5oclock_bot

    A bot that tells you a time zone where it isn't Happy Hour. Languages: Bash and Python

  7. haskell_bot

    A bot making jokes about functional programming and Haskell. Language: Haskell
